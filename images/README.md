# Папка для баннеров и картинок LinkedIn

## Как загрузить картинку

1. Открой https://github.com/YanaTimeClock365/timeclock365-landings
2. Перейди в папку `images/`
3. Нажми **"Add file" → "Upload files"**
4. Перетащи файл (JPG или PNG)
5. Нажми **"Commit changes"**

## URL картинки после загрузки

```
https://yanatimeclock365.github.io/timeclock365-landings/images/ИМЯ_ФАЙЛА.jpg
```

Пример:
```
https://yanatimeclock365.github.io/timeclock365-landings/images/banner-main.jpg
```

## Требования к картинкам для LinkedIn

- Размер: **1200 × 627 px** (идеально для LinkedIn)
- Формат: JPG или PNG
- Вес: до 5 MB
- Называй файлы без пробелов: `banner-main.jpg`, `post-access-control.jpg`
